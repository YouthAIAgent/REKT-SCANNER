## <img src="https://media.giphy.com/media/VgCDAzcKvsR6OM0uWg/giphy.gif" width="28"> `> WHAT IS THIS?`

> ```diff
> + REKT Scanner is a real-time autonomous security agent
> + that monitors EVERY contract deployed on Base mainnet.
> - It catches rug pulls, honeypots, hidden mints, and exploits
> - BEFORE they drain your wallet.
> ```

Think of it as a **paranoid security guard** watching the blockchain 24/7, backed by **AI that narrates threats** like a sports commentator reporting a live heist.

```ansi
[0;32m$ rekt-scanner --mode=live --chain=base[0m
[0;36m[MONITORING][0m Block 28491023 | 142 txs | 3 new contracts
[0;31m[THREAT]    [0m 0x7a3f...9c21 | CRITICAL RISK (87/100) | rug_pull, honeypot, hidden_mint
[0;33m[AI]       [0m "ALERT! A contract just deployed with a removeLiquidity function
              hidden behind an onlyOwner modifier on line 47..."
[0;32m[SAFE]     [0m 0xab12...ef34 | MINIMAL RISK (5/100) | Clean scan
[0;31m[REKT]     [0m Another scammer caught. Contract graveyard updated.
```
